🚀 خطوات الرفع على GitHub (مرة واحدة فقط)

1

إنشاء حساب على GitHub

افتح github.com ← Sign up ← أنشئ حساب مجاني (إذا مش عندك)

2

إنشاء Repository جديد

اضغط + ← New repository ← اسمه: gustu-menu ← اختر Public ← Create repository

3

رفع ملفات القائمة

اضغط uploading an existing file ← ارفع الملفات التالية:

📁 gustu-menu/ ├── index.html ← القائمة (للعملاء) ├── menu.json ← الأسعار والأصناف ├── admin.html ← لوحة التعديل └── qr-setup.html
4

تفعيل GitHub Pages

Settings ← Pages ← Source: Deploy from branch ← Branch: main ← Save

بعد دقيقة — رابط القائمة هيبقى:

https://YOUR-USERNAME.github.io/gustu-menu/
5

توليد QR Code من الرابط

ضع الرابط في المربع أدناه ← اضغط توليد ← اطبع أو حمّل QR Code

🔲 توليد QR Code

الرابط هو نفس الرابط الدائم — لا تحتاج QR جديد عند تحديث الأسعار. QR واحد = قائمة متحدثة دائماً.
